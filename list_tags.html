Creating a List

There are three types of lists

– Unordered Lists - ul
– Ordered Lists - ol
– Definition Lists - dl

Unordered list -ul

This list has a bullet in front of each list item.

Attributes – Type : Specifies the type of list item. For unordered list sets to DISC(solid bullet), SQUARE(solid square),CIRCLE(hollow bullet).

Ordered list

This list has a sequence of numbers in front of each list item.

Attributes – Type : Specifies the type of list item. For ordered list sets to A, a, I, i ,1
